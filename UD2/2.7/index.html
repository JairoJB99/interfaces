Elemento datalist

El elemento datalist nos permite crear una lista de objetos option que podremos reutilizar en varios elementos input sin necesidad de crear una para cada uno si va a ser la misma. Además nos permite escribir sobre el input de forma que, nos autocompletará con las opciones disponibles


        <label for="car">Select your favorite car:</label>
        <input list="cars" name="car" id="car">

        <label for="ucar">Select your car:</label>
        <input list="cars" name="ucar" id="ucar">

        <datalist id="cars">
            <option value="Renault">
            <option value="Mercedes">
            <option value="BMW">
            <option value="Audi">
            <option value="Opel">
        </datalist>

Atributo multiple

El atributo multiple del elemento input nos permite en caso de ser un input de tipo file la opción de seleccionar uno o más archivos, ya que, si lo dejamos sin el atributo por defecto solo será posible seleccionar un elemento

<input type="file" multiple />

Atributo autofocus

El atributo autofocus del elemento input nos permite que al entrar en la página el cursor se posicione en el input que tenga el autofocus así será el campo en primero escriba el usuario ya que es como si hicieramos click en él, con el atributo autofocus ya podemos escribir sobre él sin necesidad de hacer click, también resalta el contorno del input para indicar que estamos sobre él

<input type="text" placeholder="Autofocus" autofocus />
